# Olá! Eu sou o Rodrigo Ribeiro 👋

Sou um **Desenvolvedor Full Stack** apaixonado por tecnologia e focado em criar soluções eficientes e robustas, desde a infraestrutura até ao utilizador final.

### 💻 O que faço e o que estudo
